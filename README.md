# 项目介绍
此项目是一个简单的乒乓球游戏，旨在帮助用户体验基本的游戏开发概念。

# 文件架构
- `src/`: 源代码目录
  - `main.py`: 游戏主逻辑
  - `ball.py`: 球的逻辑
  - `paddle.py`: 球拍的逻辑
- `assets/`: 资源目录
  - `images/`: 游戏使用的图像文件
  - `sounds/`: 游戏音效文件

# 如何运行项目
1. 确保你安装了 Python 3.x。
2. 克隆该项目到本地：
   ```bash
   git clone https://github.com/Dracula-self/pingpong.git
   ```
3. 进入项目目录：
   ```bash
   cd pingpong
   ```
4. 运行主程序：
   ```bash
   python src/main.py
   ```
5. 开始享受游戏！
